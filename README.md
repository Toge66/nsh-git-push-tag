# nsh-git-push-tag
auto push git tag +1

参数          |      描述       
------------- | -------------  
-v      | 显示当前版本号    
-p      | production版本
-n      | noBeta版本  
-m      | minio版本
-c      | 增加备注:例如-c "first version"
-f      | 从指定文件中获取version，默认文件为:build.gradle 例如:-f "package.json"
-s      | 增加tag后缀
-h      | 帮助
